# 下個任務建議

## 功能

分析 WBS 任務清單，考慮依賴關係和優先級，建議最適合的下一個任務。

## 分析內容

1. **依賴檢查** -- 前置任務是否完成
2. **優先級排序** -- 關鍵路徑、阻塞因素
3. **複雜度評估** -- 預估時間和難度
4. **Agent 建議** -- 建議搭配的專業 Agent

## 輸出格式

```
下個任務建議:

  任務: [任務名稱]
  描述: [簡述]
  優先級: [高/中/低]
  預估: [時間]
  依賴: [前置任務（已完成）]
  建議 Agent: [agent-name]

請選擇:
  [Y] 開始此任務
  [S] 跳過，看下一個
  [D] 查看詳細資訊
  [L] 查看完整任務清單
```

## 使用方式

```
/task-next              # 取得建議
/task-next --detailed   # 含詳細分析
```

## 搭配使用

```
/task-next    → 取得任務
/plan         → 規劃實作步驟
/tdd          → 開始開發
/verify       → 完成驗證
/task-next    → 取得下一個
```
